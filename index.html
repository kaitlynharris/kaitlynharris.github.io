Web Development Skill Summary

Database Skill Summary

Something about writing SQL queries appeals to the part of my brain that has always loved puzzles. I've used both MySQL and SQLite3 with Python to create database-powered desktop apps. I've written scripts to populate and query databases in MSSQLServer. In the Portfolio section you'll find a script that generates a database for a fictional town library, complete with several tables, sample data, and a stored procedure with several possible queries.

Python Skill Summary

Who doesn't like Python? So easy to learn, so many libraries. I'm comfortable with Python2 and Python3 and have used both TKinter and wxPython to create GUIs for database-driven desktop apps. Struggling to find a name for your new band? You can find a Python band name generator in my Portfolio, as well as an app that'll create copies of all your recently-modified files.

C# & .NET Skill Summary

I started learning C# after getting comfortable with Python, and at first it felt like such a finicky language in comparison. After spending a few weeks working on a website built with Microsoft's MVC Framework, Visual Studio, Entity Framework, and SignalR, written primarily in C#, I gained a newfound understanding and appreciation for the power of the language. You can find code snippets from the website on my GitHub.

Version Control Skill Summary

If you ask me, Git is seriously cool technology. I love being able to use GitHub to store my own projects and see what my friends have been creating lately. For most of my projects, I like to use GitHub's Atom editor, iTerm, and Git command-line tools. I've also used TFS for projects in Visual Studio.
